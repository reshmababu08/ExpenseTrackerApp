# ExpenseTrackerApp
Android Expense Tracker app using Kotlin, Jetpack Compose, and Firebase Authentication.
This project allows users to securely manage daily expenses with authentication and full CRUD functionality.

---

## Features

- User Registration and Login (Email/Password and Google Sign-in using Firebase)
- Secure Firebase Authentication
- Add new expenses
- Edit existing expenses
- Delete expenses
- Category-wise expense grouping
- Total expense calculation
- Clean and responsive Jetpack Compose UI
- Loading indicators for better user experience
- Error handling for authentication and forms

---

## Tech Stack

- Kotlin
- Jetpack Compose
- Firebase Authentication
- MVVM Architecture
- ViewModel
- State Management using Compose

## App Screens

- Splash Screen

---

## Firebase Setup

1. Create a Firebase project in Firebase Console
2. Add Android application in Firebase
3. Enable Authentication methods:
   - Email/Password
   - Google Sign-in
4. Download google-services.json file
5. Place it inside the app directory

---

## How to Run the Project

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the application on an emulator or physical device

## Generate APK

Build → Build Bundle(s) / APK(s) → Build APK(s)

APK will be generated at:
app/build/outputs/apk/debug/app-debug.apk
